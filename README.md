# Shiva Website

Static website for Shiva Bhandari / 369 Jyotish.

## Main Files

- `index.html` - public website
- `admin.html` - admin dashboard
- `Moon.png` - moon image used in the hero section
- `shiva-profile.jpg` - profile image
- `full-moon.jpg` - alternate moon texture
- `CNAME` - custom domain config for GitHub Pages

## Local Preview

You can open the site with VS Code Live Server or any static file server.

## Deployment

This repo is set up to be pushed to GitHub and can be served with GitHub Pages or another static host.
